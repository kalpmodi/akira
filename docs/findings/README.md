# Findings Archive

Detailed finding writeups from real bug bounty engagements using Akira.

See [FINDINGS.md](../../FINDINGS.md) for the full live findings table.

This directory will contain per-finding detailed reports as they are cleared for public disclosure.

## Format

Each finding gets its own file: `YYYY-MM-finding-title.md`

Structure:
- Target type (anonymized)
- Skill chain used
- Step-by-step reproduction
- HTTP evidence
- Business impact
- Remediation
